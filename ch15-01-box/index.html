    Boxes donâ€™t have performance overhead, other than storing their data on the heap instead of on the stack. But they donâ€™t have many extra capabilities either. Youâ€™ll use them most often in these situations:

    More Information About the Cons List

    A cons list is a data structure that comes from the Lisp programming language and its dialects and is made up of nested pairs, and is the Lisp version of a linked list.     (1, (2, (3, Nil)))

    Each item in a cons list contains two elements: the value of the current item and the next item. The last item in the list contains only a value called Nil without a next item.
